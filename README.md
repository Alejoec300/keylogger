Keylogger en Python

Este proyecto consiste en la implementación de un **keylogger** básico desarrollado en Python utilizando la librería keyboard. El objetivo principal es comprender cómo se interceptan los eventos del teclado a nivel de sistema operativo y cómo se gestionan los archivos de registro (logs).

Este software ha sido creado exclusivamente con fines educativos y de investigación en seguridad informática. El uso de este código para monitorear equipos sin autorización es ilegal y poco ético.

Tecnologías y Requisitos

* **Lenguaje:** Python 3.10+
* **Librerías:** keyboard
* **Entorno:** Se recomienda el uso de un entorno virtual (venv).
* **Permisos:** Requiere privilegios de Administrador (Windows) o Root (Linux).

---

Estructura del Proyecto

* main.py: Código fuente principal que contiene la lógica de captura y guardado.
* requirements.txt: Lista de dependencias necesarias.
* log.txt: Archivo generado automáticamente donde se almacenan las pulsaciones.
